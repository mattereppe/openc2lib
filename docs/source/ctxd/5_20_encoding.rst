5.20 Encoding
=============

This data type defines the encoding format to be used. Other encodings
are permitted, the type Encoding can be extended with other encoders
(e.g., XML).

Type: Encoding (Enumerated)

+---+--------+------------------------------------+
| I | Name   | Description                        |
+===+========+====================================+
| 1 | json   | JSON encoding                      |
+---+--------+------------------------------------+
