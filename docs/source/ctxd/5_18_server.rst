5.18 Server
===========

It specifies the hostname or the IPv4 address of a server.

Type: Server (Choice)

+---+----------+---------+---+-------------------------------------------+
| I | Name     | Type    | # | Description                               |
| D |          |         |   |                                           |
+===+==========+=========+===+===========================================+
| 1 | hostname | Hostname| 1 | Hostname of the server                    |
+---+----------+---------+---+-------------------------------------------+
| 2 | ipv4-addr| IPv4-Addr| 1 | 32-bit IPv4 address as defined in         |
|   |          |         |   | [RFC0791]                                 |
+---+----------+---------+---+-------------------------------------------+
